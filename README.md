# Janus
To do list for short- and long-term goals made in typescript.

Actually Janus is in development, so that don't have useful features yet. All data is storaged in memory.

## How to use?
clone this repository
```bash
git clone git@github.com:kei4ss/Janus.git
```

install npm packages
```bash
npm install
```

execute the project
```bash
npm run build
npm run start
```
